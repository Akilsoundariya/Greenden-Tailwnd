# 🌿 Greenden - Tailwind CSS Website

Greenden is a responsive website built using HTML and Tailwind CSS.  
This project focuses on creating a clean, modern layout using utility-first styling.

## 🛠️ Technologies Used
- HTML5
- Tailwind CSS
- CSS3


## ✨ Features
- Fully responsive design
- Modern UI layout
- Utility-first CSS styling
- Clean and structured code
- Mobile-friendly interface
